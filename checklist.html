✅ 出國檢查清單

出發前、整理行李時、前往機場前都可以快速確認一次

🪪

1. 證件

  • 護照
  • 簽證 / 入境許可
  • 身分證
  • 機票 / 電子機票
  • 住宿預訂確認
  • 旅遊保險資料
  • 行程表
  • 護照影本 / 電子備份
  • 緊急聯絡資訊
🚆

2. 交通

  • 機場接送安排
  • 當地交通卡 / 車票
  • 租車預約資料
  • 國際駕照
  • 地圖 / 離線地圖
  • 航班時間再次確認
📶

3. 通訊

  • 手機
  • SIM 卡 / eSIM
  • 漫遊方案確認
  • 充電線
  • 聯絡人資訊備份
  • 翻譯 App
  • 通訊軟體可正常使用
🔌

4. 電器

  • 充電器
  • 行動電源
  • 萬國轉接頭
  • 延長插座
  • 耳機
  • 筆電 / 平板
  • 相機 / 記憶卡 / 備用電池
💊

5. 藥品

  • 常備藥
  • 處方藥
  • 暈車 / 暈機藥
  • 止痛藥
  • 腸胃藥
  • OK 繃
  • 口罩
  • 酒精 / 消毒用品
💴

6. 金錢

  • 現金（當地貨幣）
  • 信用卡
  • 金融卡
  • 外幣兌換完成
  • 錢包
  • 小額零錢
  • 銀行海外刷卡 / 提款通知設定
🧳

7. 行李

  • 行李箱 / 背包
  • 行李吊牌
  • 衣物
  • 內衣褲
  • 襪子
  • 外套
  • 睡衣
  • 鞋子 / 拖鞋
  • 雨具
  • 盥洗用品
  • 毛巾
  • 保養品 / 化妝品
  • 洗衣袋 / 收納袋
📝

8. 其他

  • 水壺
  • 衛生紙 / 濕紙巾
  • 旅遊枕
  • 墨鏡
  • 帽子
  • 防曬用品
  • 行程需要的門票
  • 禮物 / 伴手禮
  • 家中電器與門窗確認
  • 請假 / 寵物 / 信件代管安排
